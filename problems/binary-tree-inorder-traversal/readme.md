# Binary Tree Inorder Traversal

Given a binary tree, output its elements as of priorder traversal. Recursive solution is trivial, so
try to use iterative one. 

Example
-------

Given this binary tree:

```text
   1
    \
     2
    /
   3
```

return `[1,3,2]`.
