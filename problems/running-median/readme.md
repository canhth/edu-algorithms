# Running Median

You are given an array of integers. Return the N numbers which represent the medians of the sorted 
arrays formed from the 0..N array on each step.

Example
-------

Example: `[100, 2, 3, 4, 6, 1]` -> `[100, 51, 3, 3.5, 4, 3.5]`.
