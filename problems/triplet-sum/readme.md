# Triplet Sum

Given an array of numbers, find the first three numbers which sum equals to a given `S`.

Example
-------

Given `[1,2,3,4]` and `S = 6`, return `[1,2,3]`. When `S = 1` return `null` as there's no 3 elements
which sum is `6`.
