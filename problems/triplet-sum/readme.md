# Pair Sum

Given an array of numbers, find the first two numbers which sum equals to a given `S`.

Example
-------

Given `[1,2,3,4]` and `S = 5`, return `[2,3]`. When `S = 1` return `null` as there's no 2 elements
which sum is `5`.
